# 黄敬杰 Portfolio

这是黄敬杰的作品集网页项目，可用于 GitHub Pages 部署。

## 文件结构

- `index.html`：作品集首页
- `assets/`：网页图片与压缩后的网页视频
- `resume/`：简历文件

## 部署到 GitHub Pages

1. 新建 GitHub 仓库，例如 `portfolio-site`
2. 上传本文件夹内所有内容
3. 在仓库 Settings → Pages 中选择 `Deploy from a branch`
4. Branch 选择 `main`，目录选择 `/root`
5. 保存后等待生成访问链接

## 注意

GitHub 不适合直接存放 100MB 以上的大视频。当前版本已将大话西游完整版和第五人格 ep01 压缩为网页可播放的 `.webm` 文件，当前轻量上传版已将主要视频压缩到 10MB 以下，更适合通过 GitHub 网页端上传。
